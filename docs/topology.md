# Topology

Use generic names when publishing public results.

```text
CONTROL_HOST
├── CLIENT_A
├── CLIENT_B
├── CLIENT_C
└── SERVER_A
```

## Suggested private-to-public naming

| Private value | Public alias |
|---|---|
| baxter-debian | SERVER_A |
| lenovo-debian | CLIENT_A |
| dell-ubuntu | CLIENT_B |
| dexter-debian | CLIENT_C |

Do not publish public IPs, internal notes, credentials, or private LAN details unless intentionally sanitized.
